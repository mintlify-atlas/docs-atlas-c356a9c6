## What is AiVault?

AiVault is a curated directory of powerful AI tools, updated daily. Whether you're looking for AI writing assistants, image generators, coding tools, or specialized solutions, AiVault helps you find exactly what you need.

With over **40+ curated AI tools** from industry leaders like OpenAI, Anthropic, Google, GitHub, Adobe, Notion, Canva, and Figma, AiVault provides a comprehensive resource for discovering and managing AI tools.

## Tech Stack

AiVault is built with modern, production-ready technologies:

- **Frontend:** Next.js 15+ (App Router, React 19)
- **Styling:** Tailwind CSS 4 & shadcn/ui
- **Backend/Database:** Convex (Reactive Serverless Database)
- **Auth:** Clerk (JWT-based session management)
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
- **Emailing:** Resend (Custom transactional templates)

<Tip>
  AiVault is designed with a **Security-First** mindset, featuring server-side validation, RBAC (Role-Based Access Control), input sanitization with Zod schemas, and CSRF & XSS protection.
</Tip>
