The AiVault directory makes it easy to discover and explore over 40 curated AI tools across multiple categories.

## Viewing the Directory

Navigate to the Tools page to browse all available AI tools. Tools are displayed in a responsive grid layout with key information at a glance.

### Tool Cards

Each tool is presented in a card that displays:

- **Logo**: Visual identifier for the tool (or first letter if no logo is available)
- **Tool Name**: Prominently displayed at the top
- **Description**: Short one-liner explaining what the tool does
- **Category**: Classification like Writing, Image, Video, Coding, Productivity, Audio, or Design
- **Pricing Model**: Free, Freemium, or Paid
- **Tags**: Up to 3 relevant tags for quick identification
- **Upvotes**: Community popularity rating
- **Badges**: Special indicators for New or Featured tools

<Note>
Featured tools have been highlighted by our team for their exceptional quality or unique capabilities.
</Note>

## Exploring Categories

AiVault organizes tools into 8 main categories:

- **Writing**: Content creation, copywriting, and text generation tools
- **Image**: Image generation, editing, and enhancement tools
- **Video**: Video creation, editing, and processing tools
- **Coding**: Development assistance, code generation, and debugging tools
- **Productivity**: Task management, automation, and workflow tools
- **Audio**: Voice synthesis, music generation, and audio processing
- **Design**: UI/UX design, graphic design, and creative tools

### Category Filtering

<Tabs>
  <Tab title="Desktop">
    On desktop, category filters appear in the left sidebar. Click any category to filter tools instantly.
    
    Select "All" to view all available tools across categories.
  </Tab>
  <Tab title="Mobile">
    On mobile devices, categories appear as horizontal scrollable chips below the page title.
    
    Swipe left or right to view all categories, then tap to filter.
  </Tab>
</Tabs>

## Viewing Tool Details

Click on any tool card to view its full details page, which includes:

- Complete description and overview
- Full list of features and capabilities
- Use cases and applications
- Pros and cons analysis
- Platform availability (Web, iOS, Android, Desktop, API, Chrome Extension)
- Social links (Twitter, GitHub, Discord)
- Community upvotes and engagement

<Tip>
Tool detail pages also show related tools in the same category to help you discover alternatives.
</Tip>

## Sort Options

Control how tools are ordered using the sort dropdown:

- **Newest Arrivals**: Shows recently added tools first (default)
- **Most Popular**: Sorts by upvote count to surface community favorites

## Active Filters Display

When you apply filters or search, an active filters section appears below the search bar showing:

- Current category filter
- Current pricing filter
- Active search query

Click the X icon on any filter badge to quickly remove it.

<Tip>
Use the "Clear All Filters" button to reset all filters and return to viewing all tools.
</Tip>
