AiVault provides multiple ways to narrow down and find the perfect AI tool for your needs.

## Search

The search bar allows you to find tools by name, description, or tags.

### What You Can Search

The search engine looks across:

- **Tool names**: Exact or partial matches (e.g., "GPT" finds ChatGPT, GPT-4, etc.)
- **Descriptions**: Keywords in short or long descriptions
- **Tags**: Any tags associated with tools

### How to Search

1. Enter your search term in the search bar
2. Results update automatically as you type
3. Search is case-insensitive for better matching

<Tip>
Search works best with specific keywords. Try searching for technologies ("Stable Diffusion"), use cases ("writing"), or features ("API").
</Tip>

### Search Examples

- `chatbot` - Finds all conversational AI tools
- `image generation` - Discovers tools for creating images
- `free` - While not a filter, finds tools mentioning free in descriptions
- `coding assistant` - Locates development and programming tools

## Category Filters

Filter tools by their primary category:

- **All**: Shows all tools (default)
- **Writing**: Content creation and text generation
- **Image**: Image generation and editing
- **Video**: Video creation and processing
- **Coding**: Development and programming tools
- **Productivity**: Workflow and automation
- **Audio**: Voice and music generation
- **Design**: UI/UX and graphic design

### Using Category Filters

<Tabs>
  <Tab title="Desktop">
    Categories appear in the left sidebar:
    
    1. Click any category name to filter
    2. The active category is highlighted
    3. Tool count updates to show filtered results
    4. Click "All" to remove the category filter
  </Tab>
  
  <Tab title="Mobile">
    Categories appear as horizontal chips:
    
    1. Swipe left/right to see all categories
    2. Tap a category to filter
    3. Active category is highlighted with distinct styling
    4. Tap "All" to show all tools again
  </Tab>
</Tabs>

## Pricing Filters

Narrow results by pricing model:

- **All**: Shows all pricing models (default)
- **Free**: Tools that are completely free to use
- **Freemium**: Tools with free tier and paid upgrades
- **Paid**: Tools that require payment to use

<Note>
Pricing filters help you quickly find tools that fit your budget. Freemium tools often provide generous free tiers.
</Note>

### Desktop Pricing Filter

On desktop, pricing options appear in the left sidebar below categories:

1. Click any pricing option to filter
2. Active filter is highlighted
3. Click "All" to remove the pricing filter

### Mobile Pricing Filter

On mobile, access pricing filters from the Filters sheet:

1. Tap the "Filters" button in the top-right corner
2. The bottom sheet slides up showing all filter options
3. Select your desired pricing option
4. Results update automatically
5. The Filters button shows a count badge of active filters

## Combining Filters

All filters work together for precise results:

```
Category: "Coding" + Pricing: "Free" + Search: "API"
= Free coding tools with API capabilities
```

### Active Filters Display

When filters are active, a results summary appears showing:

- Each active filter as a removable badge
- "Results for:" label indicating filtered view
- Quick remove (X) button on each badge

Click the X on any badge to remove just that filter, or use "Clear All Filters" to reset everything.

## Sort Options

Control the order of results:

### Newest Arrivals (Default)

- Shows recently added tools first
- Great for discovering new additions
- Based on tool submission date

### Most Popular

- Sorts by upvote count (highest first)
- Surfaces community favorites
- Best for finding proven, well-liked tools

<Tip>
Try "Most Popular" when exploring a new category to see the community's top picks.
</Tip>

## Reset Filters

Clear all active filters and return to the full directory:

- **Desktop**: Click "Reset All Filters" button in the sidebar
- **Mobile**: Click "Clear All Filters" in the empty state, or reset individual filters in the Filters sheet
- **Active Filters**: Click X on each filter badge, or use the clear button

## No Results

If your search and filters return no results:

1. A message appears: "No tools found"
2. Suggestion to "Try adjusting your search or filters"
3. "Clear All Filters" button to quickly start over

<Note>
Very specific filter combinations may yield no results. Try broadening your search by removing one or more filters.
</Note>

## Performance Tips

- Search updates as you type for instant results
- Filters apply immediately without page reload
- All filtering happens client-side for fast response
- Results animate smoothly when filters change

Bookmark frequently used filter combinations by saving the URL - category and search parameters are included in the page URL.
