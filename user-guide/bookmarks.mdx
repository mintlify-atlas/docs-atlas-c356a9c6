Bookmark tools you want to remember or try later. Your bookmarks are private and accessible from your dashboard.

## Requirements

<Note>
You must be signed in to bookmark tools. Bookmarks are tied to your user account.
</Note>

## Adding Bookmarks

Bookmark any approved tool from the directory:

1. Navigate to a tool's detail page
2. Click the bookmark icon or button
3. The tool is instantly added to your bookmarks

<Tip>
The bookmark icon changes appearance when a tool is bookmarked, giving you instant visual feedback.
</Tip>

## Viewing Your Bookmarks

Access your bookmarked tools from your user dashboard:

- View all tools you've saved in one place
- Bookmarks display the same information as directory tool cards:
  - Logo and name
  - Description
  - Category and pricing
  - Tags and upvotes
  - New and Featured badges

## Managing Bookmarks

### Removing Bookmarks

Unbookmark a tool using either method:

- From the tool's detail page: Click the bookmark icon again
- From your bookmarks view: Click on the tool to visit its page, then unbookmark

### Organizing Your Collection

Bookmarks are displayed in the order you added them, with most recent first. Use bookmarks to:

- Create a personal collection of tools you use regularly
- Save tools to evaluate later
- Build a curated list for a specific project or use case
- Keep track of tools you want to recommend to others

## Bookmark Features

### Instant Toggle

The bookmark system uses real-time updates:

- Bookmarking is instant with no page reload
- Changes sync across all your devices
- Visual feedback confirms your action

### Persistent Storage

Your bookmarks are:

- Stored securely in the database
- Accessible from any device where you're signed in
- Preserved even if tools are updated

### Privacy

Your bookmarks are completely private:

- Only you can see which tools you've bookmarked
- Bookmarks don't affect public tool rankings
- Other users cannot view your bookmark collection

<Note>
Bookmarks are separate from upvotes. Bookmarking a tool doesn't automatically upvote it.
</Note>

There is no limit to the number of tools you can bookmark. Build as large a collection as you need.

</Tip>
